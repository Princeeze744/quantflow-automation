## Features

**Trade Journal** — Log trades manually or CSV import (MT4/MT5/cTrader). TradingView charts with Entry, Exit, SL, TP lines. Psychology check-in, grading (A-F), reflections. Edit trades with auto P&L recalculation. Branded PNG share cards.

**Analytics Engine** — Win rate, profit factor, expectancy, max drawdown. Equity curve, P&L calendar, heatmap, pair breakdown, session analysis, win/loss streaks.

**Trading Psychology** — Tiltmeter discipline gauge tracking consecutive losses, revenge trades, FOMO, and mood. What-If Simulator with 4 scenarios.

**Strategy Playbooks** — Define entry rules, exit rules, confluences. Track performance per playbook.

**Risk Manager** — Position calculator for 60+ instruments (forex, metals, energy, crypto, indices). Drawdown tracker, growth projector.

**Market Intelligence** — Live session clock, currency strength, economic calendar, correlation matrix, news feed.

**AI Coach** — Automated insights, chat interface, weekly reviews. Powered by Trade2Retire Academy.
